Purpose of this sample is to test custom extension loading to DAS 4.0.0

1. Copy {WSO2DASHome}/samples/artifacts/0033/timeSeriesExtensionSample.siddhi file to {WSO2DASHome}/deployment/siddhi-files

2. Start the worker using ./{WSO2DASHome}/bin/worker.sh

3. Run {WSO2DASHome}/samples/artifacts/0033/publisher.sh using the command :
    bash publisher.sh

4. See the output in the WSO2DAS terminal
