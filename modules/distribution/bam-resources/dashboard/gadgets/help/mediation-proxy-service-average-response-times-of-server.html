Average Processing Times of Proxy Services Gadget

Introduction

This gadget shows the average processing time of each proxy service. The graph updates for every 30 seconds and the time is presented over the X-axis (in seconds) where the corespondent average processing time is presented over the Y-axis (in milliseconds). This chart shows history data up to 270 seconds. Advanced tools available in the "canvas" (maximized) mode.

Configuration

Please click on the configuration tab and select the server to monitor from the drop down.
