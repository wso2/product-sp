Server Drill Down Gadget

Introduction

This gadget shows the realtime information such as available ESB servers, available endpoints for a given server and number of faults/requests for a given endpoint. Simply point the mouse over the graph to see extended information.

Servers Tab

Shows realtime a wedge graph of all available ESB servers.

Endpoints Tab

Shows realtime a wedge graph of endpoints of the selected server. Use the dropdown to select the server.

Faults Tab

Shows realtime linear graph of faults and requests of a given endpoint. Use the dropdown to select the endpoint.

Selection using mouse click on the graph (experimental Feature)..

You can click on the graph to select a server in Servers Tab and to select a endpoint in Endpoints tab in Firefox/Google Chrome browsers. IE is not supported yet for this feature
